Summary

This is the first stable public full-v1 release of the Paired Engineering body of work.

It packages the framework, leadership material, workshop layer, practice layer, evidence kit, reference deployment guidance, and generated slideware into a public repo structure intended to stand on its own.

Included in this release

  • public positioning, one-pager, getting-started, and adopter-path artifacts
  • core framework and rollout material
  • leadership notes and deck-ready markdown
  • workshop pack, exercise library, worksheet packs, and scenario cards
  • pilot evidence kit and operations material
  • Microsoft Forms, Excel or SharePoint, and Jira reference deployment notes
  • generated public-safe PowerPoint exports with speaker notes and baseline styling

Release posture

  • passes both the public-core and full-v1 bundle audits
  • organized by use case and release bundle

Slide deck exports

Included in slides/:

  • Executive Deck - AI Enablement for an Initial Pilot Cohort A concise leadership deck for sponsoring a bounded initial pilot and understanding the core operating stance.
  • Public-Facing Deck - AI Enablement A public overview of the problem, the stance, and the operating model behind Paired Engineering.
  • Leadership Reference Deck - AI Enablement for an Initial Pilot Cohort A modular follow-up deck for deeper leadership questions once the core conversation is underway.
  • Manager Enablement Deck - Leading AI Adoption in Delivery Teams A management-facing deck for team conditions, learning protection, and practical rollout signals.
  • Staff Engineer Enablement Deck - AI Enablement Through Influence A technical-leadership deck for standards, workflow design, verification, and influence without authority.
  • Practitioner Workshop Deck - Paired Engineering with AI A workshop-oriented practitioner deck for paired-engineering habits and verification behavior.
  • Internal Advocacy Deck - Paired Engineering with AI A lighter communication deck for internal socialization and local AI enablement advocacy.

Known limitations

  • the current PPTX styling is a reusable baseline, not yet a fully branded PowerPoint template
