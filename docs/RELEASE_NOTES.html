Summary

This release promotes the approved v2 language architecture to the public package for Paired Engineering.

The core framework, workshop layer, evidence kit, and slideware are still here, but the public surface now presents them through a clearer naming hierarchy, tighter category definition, and more consistent bundle titles.

Why this is a major release

This release changes the public interface layer of the project:

  • the category phrase is now A practical delivery model for AI-enabled software teams.
  • public-facing artifact titles have been harmonized around the Paired Engineering naming system
  • executive, leadership, manager, and Staff deck families now use updated public names
  • the generated HTML review and Pages-facing copy now follow the same language architecture

These are not just wording tweaks. They change how the package is understood, navigated, linked, and repeated, which is why this release is versioned as 2.0.0.
