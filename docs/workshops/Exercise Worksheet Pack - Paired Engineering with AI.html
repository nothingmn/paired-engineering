Purpose

This note turns the exercise library into facilitator-ready worksheet packs.

The exercise notes remain the canonical scenario cards.

These worksheet packs add the teaching layer:

  • recommended audience
  • timebox
  • facilitation mode
  • setup notes
  • learner instructions
  • what good looks like
  • likely traps
  • debrief prompts

How to use this pack

  1. Choose the learner track that best matches the audience.
  2. Pick an exercise whose size and complexity fit the session.
  3. Use the worksheet pack for delivery structure.
  4. Use the canonical exercise note as the scenario source of truth.
  5. Capture what worked so the exercise difficulty model can be calibrated later.

Facilitation rule

The goal is not to see who can prompt fastest.

The goal is to see whether participants can:

  • choose the right mode
  • use AI deliberately
  • verify appropriately
  • resist false confidence
  • explain what they trust and why

Recommended next refinement

After one or two real pilots, capture:

  • actual completion time by exercise
  • where learners became overconfident
  • where the size and complexity scores felt too high or too low
  • which worksheet prompts created the strongest reflection
