# Support

## Where to ask for help

- use GitHub Issues for bugs, broken links, export problems, and packaging defects
- use GitHub Discussions, if enabled, for broader questions, ideas, and interpretation

## Helpful issue details

- what you were trying to use
- the file or artifact involved
- the deck name and slide number, if the issue is in slideware
- the app you used, such as PowerPoint, Keynote, or Google Slides
- screenshots or error output when relevant

## A practical note about artifacts

- generated slide decks are built from markdown source notes
- content fixes usually belong in the source notes first, then in regenerated outputs
- visual rendering issues may depend on the app being used
